== OpenSeadragon Image Viewer ==

This plugin adds a zoomable image viewer, using  the open source [OpenSeadragon Image Viewer](https://openseadragon.github.io/).

It allows for tiled image sources and normal jpg, png images to be displayed inside the viewer.

== Features ==

* Can show the feature image
* Can show a preconfigured image from the site editor
* Adjustable image viewer height in the settings of the block
